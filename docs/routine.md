## Strength Workouts

Rules across all sessions:
- 2-3 sets, 8-10 reps
- 60-90 sec rest between sets
- Breathe normally — never hold breath
- Start light, increase slowly
- Stop if dizzy, chest pain, or strong headache

## Walk/Run Program — Building to 10K (12 Weeks)

3 sessions per week at 4pm (Mon / Wed / Fri).

| Week  | Each session | Notes |
|-------|-------------|-------|
| 1–2   | Walk 20 min, run 1 min every 5 min | Easy, just get moving |
| 3–4   | Run 2 min / walk 3 min × 6 | Build the rhythm |
| 5–6   | Run 5 min / walk 2 min × 4 | First real running |
| 7–8   | Run 10 min / walk 1 min × 3 | Build endurance |
| 9–10  | Run 15 min / walk 1 min × 2 + long run 35-40 min Fri | Near 5K |
| 11    | Run 20 min continuous + long run 45 min Fri | 5K territory |
| 12    | Run 25 min + 10K attempt Fri | Goal race |

**Key rules:**
- Pace: conversational — if you can't talk, slow down
- Never run the day after heavy leg strength if knees are sore
- If morning BP is consistently above 140/90, walk only that week
- Increase long run by max 10% per week

---

## Blood Pressure Tracking

- Check every morning before coffee, same time, same arm
- Log it — look for the trend, not individual readings
- Target: diastolic below 80
- If readings spike for 3+ days in a row, reduce exercise intensity that week

## Simple Rules

- Protect the 4pm slot like a meeting
- Consistency over intensity — moderate and steady beats hard and sporadic
- Weekend activity with kids counts as active recovery
- Night work (9-11pm) = light tasks only, no deep focus
- Don't chase soreness — the goal is showing up every week for 6 months
